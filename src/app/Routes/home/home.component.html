Create and connect to the API Service.
  1. ng generate service Services/API
  2. Add "private http: HttpClient" to the constructor.
  3. Copy the UserIdentktyModel.ts and YouTubeVideoModel.ts files into the Models directory.
  4. Add the "getUserIdentity" method to api.service.ts.
  5. Add the code in app.component.ts to call and subscribe to the "getUserIdentity" method in api.service.ts.
  6. In app.module.ts, add HttpClientModule as an import, and APIService as a provider.
  7. Note the fact that the addition of the http call causes an error in Karma.
